# 🏒 Winnipeg Jets Hub  
A clean, modern, Jets‑themed hockey website built with HTML, CSS, and JavaScript.  
This project includes roster information, news, schedules, results, trades, and player statistics — all styled to match a professional NHL‑style layout.

---

## 🚀 Features

### 🔹 Homepage
A simple landing page that links to each section of the site.

### 🔹 Roster
- Organized by position (Forwards, Defensemen)
- Clean horizontal roster cards with position badges
- Projected game lines displayed in a 2‑1 formation

### 🔹 News
- Simple news cards for team updates

### 🔹 Schedule
- Horizontal schedule rows with Jets‑themed styling
- Includes opponent, date, and game type (Home, Road, Regular)

### 🔹 Results
- Horizontal result boxes with:
  - Thick colored left border (green = win, red = loss)
  - Date, opponent, score, and location
  - Clean NHL‑style layout

### 🔹 Stats
- Skater stats table
- Goaltender stats table
- Automatic stat leader calculation for:
  - Goals
  - Assists
  - Shots on Goal
  - Plus/Minus
  - Pokes
  - Hits
  - Wins (Goalies)
  - Save Percentage
  - GAA
  - Shutouts
